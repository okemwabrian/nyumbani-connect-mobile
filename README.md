# Nyumbani Connect Mobile App 💙

Nyumbani Connect is a mobile platform built to connect verified house managers with trusted agents and employers across Nairobi and the 47 counties of Kenya. 

This repository contains the Flutter mobile client, which serves as the primary interface for Workers and Employers on the go, complementing the React web dashboard for Agents.

## ✨ Key Features
* **Role-Based Onboarding:** Seamless registration flows tailored for three distinct user types: House Managers (Workers), Employers, and Agents/Bureaus.
* **Strict Age Validation:** Built-in date-of-birth calculators during registration to strictly enforce an 18+ age requirement for all house managers.
* **Unified Branding:** A responsive, deep-blue UI that perfectly mirrors the Nyumbani web platform for a consistent cross-platform user experience.
* **Cross-Platform:** Compiles to both Android and iOS from a single Dart codebase.

## 🛠 Tech Stack
* **Frontend:** Flutter & Dart
* **Architecture:** Feature-first folder structure
* **Backend (Planned):** Django REST Framework (DRF) with MySQL
* **Real-Time Sync (Planned):** Django Channels & WebSockets
